Analyzing Content

Analyze the content of my Instagram posts. The dataset has two columns, namely caption and hashtags, which will help understand the kind of content posted on Instagram. Create a wordcloud of the caption column and hashtags column to look at the most used words and hashtags in the caption of Instagram posts

Python Code 
#Iterate through the Caption column of the DataFrame data (a pd.Series) 
#and concatenate all captions into one big string, separated by spaces " "
text = " ".join(i for i in data.Caption)
stopwords = set(STOPWORDS) #Get the default stopwords set of the wordcloud library ( ex: the, and)
# Create a WorldCloud without stopwords (stopwords= stopwords)
# generate(text) will separate word and count freqency, sorting, arrange text in image by frequency
wordcloud = WordCloud(stopwords=stopwords, background_color="white").generate(text)
plt.style.use('classic')
plt.figure( figsize=(12,10))
plt.imshow(wordcloud, interpolation='bilinear') #'bilinear': smoothing when displaying (anti-aliasing)
plt.axis("off")#Hide axes/coordinate frames
plt.show()

Instagram Reach Prediction Model

Python Code 
#split the data into training and test sets before training the model
x = np.array(data[['Likes', 'Saves', 'Comments', 'Shares', 
                   'Profile Visits', 'Follows']])
y = np.array(data["Impressions"])
xtrain, xtest, ytrain, ytest = train_test_split(x, y, 
                                                test_size=0.2, 
                                                random_state=42)
Python Code 
#train a machine learning model to predict the reach of an Instagram post using Python:
model = PassiveAggressiveRegressor()
model.fit(xtrain, ytrain)
model.score(xtest, ytest)
Outputs
0.7943006608607764
Python Code 
#predict the reach of an Instagram post by giving inputs to the machine learning model
# Features = [['Likes','Saves', 'Comments', 'Shares', 'Profile Visits', 'Follows']]
features = np.array([[283.0, 235.0, 5.0, 9.0, 165.0, 54.0]])
prediction = model.predict(features)[0] 
print(f"Predict the reach of an Instagram post: {prediction:,.0f}")
